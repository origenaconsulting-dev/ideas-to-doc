# ideas-to-doc
Turn a topic and loose ideas into a structured, polished Word document.

Use this skill whenever the user provides a topic and a set of ideas, notes, or thoughts they want turned into a structured Word document (.docx).

Triggers include: "organise my ideas", "turn these notes into a document", "structure my thoughts", "make a document from my ideas", "help me write this up", "draft a briefing note", "turn this into a memo", or any time the user provides a topic + raw ideas and wants a polished written output.

Also trigger when the user gives a brain dump and asks for order, structure, or a first draft — even in professional, consulting, policy, or research contexts.

Do NOT use for purely conversational requests or when no document output is needed.
